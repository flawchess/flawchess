## Sampling Rate

- **After every task commit:** Run the quick command for the affected stack (backend or frontend).
- **After every plan wave:** Run the full suite for the affected stack(s).
- **Before `/gsd-verify-work`:** Both full suites must be green AND `ty check` + `ruff check` + frontend `npm run lint` clean (pre-PR checklist from CLAUDE.md).
- **Max feedback latency:** ≤ 45 seconds for the quick command after any task commit.
