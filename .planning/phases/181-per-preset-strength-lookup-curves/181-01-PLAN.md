<objective>
Build the shipping per-preset bot-strength lookup artifact: a new stdlib-only Python generator
that reads the frozen Phase-180 sweep output (reports/data/bot-curves-internal-scale.json),
fits each preset's `internal_rating = f_preset(bot_elo)` via hand-rolled PAVA isotonic
regression, converts to approximate human-blitz ELO via `internal − G_preset_combined + C`,
inverts into per-preset `target_blitz_elo → bot_elo` lookups in 100-ELO steps with honest
inward-rounded ranges, and emits both a components+derived JSON artifact and a CI-drift-checked
generated TS module (with the canonical approximate-ELO disclaimer constant).

Purpose: single source of truth for all future labeled bot strength claims (custom bot builder,
preset cards, SEED-098 personas). No consumer imports it yet.
Output: scripts/gen_bot_strength_curves.py, reports/data/bot-strength-lookup.json,
frontend/src/generated/botStrengthCurves.ts, tests, plus CI drift step + knip ignore wiring.
</objective>
