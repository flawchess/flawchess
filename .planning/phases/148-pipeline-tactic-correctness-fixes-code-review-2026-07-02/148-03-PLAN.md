<objective>
Fix item 3 (D-06 seam: stats covariance = item 3): the quintile significance test
treats overlapping cohorts as independent, dropping the anti-correlated covariance
term and producing false "significant" verdicts.

Per D-04 (covariance-correction, NOT a full paired test — least invasive): track the
shared-game count `m` per (tc, quintile) where `user_quintile == opp_quintile`, and add
`+2m·v/(n_u·n_o)` to the SE inside the existing `compute_score_difference_test`.
Point estimates are unaffected. Also fix the wrong independence docstring.

The algebra is numerically validated against CONTEXT.md's own 100/100/100 worked
example: buggy SE 0.0707 → corrected SE 0.1000 exactly (RESEARCH.md § "Item 3 — D-04
covariance algebra").

Purpose: shared-cohort quintile comparisons stop over-reporting significance.
Output: threaded shared-count through 2 files, docstring fix, new + updated tests.
</objective>
