<objective>
Make `/bots` a real, reachable, never-locked top-level page (PLAY-01).

The `/bots` route already exists and is already lazy-loaded; it is simply unlinked from the nav. This plan
adds the nav entries (desktop + mobile bottom bar + more-drawer), places Bots second, and exempts it from
the import lock ‚Äî guests and zero-game users are precisely the audience for free bot play, so the item must
never dim.

The lock rule is duplicated in THREE places with three slightly different exemption lists. Patching one and
copy-pasting is the failure mode this plan exists to prevent, which is why it also creates the codebase's
first `App.test.tsx`.

Purpose: PLAY-01, and a regression net under a nav that has never had one.
Output: nav entries in all three surfaces, plus an RTL test that pins them across every `navUnlocked` state.
</objective>
