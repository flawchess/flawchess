  <name>Task 1: Backend stats schemas, repository, service, and router</name>
  <files>app/schemas/stats.py, app/repositories/stats_repository.py, app/services/stats_service.py, app/routers/stats.py, app/main.py</files>
  <behavior>
    - GET /stats/rating-history returns { chess_com: RatingDataPoint[], lichess: RatingDataPoint[] } where each point has { date: str, rating: int, time_control_bucket: str }
    - GET /stats/rating-history?recency=month returns only games within the last 30 days
    - GET /stats/global returns { by_time_control: WDLByCategory[], by_color: WDLByCategory[] } where each has { label, wins, draws, losses, total, win_pct, draw_pct, loss_pct }
    - GET /stats/global?recency=year filters to last 365 days
    - Both endpoints return 401 without auth token
    - Rating history excludes games where user_rating is None
    - Global stats exclude games where time_control_bucket is None (for by_time_control)
  </behavior>
  <action>
    1. Create `app/schemas/stats.py` with Pydantic v2 models:
       - `RatingDataPoint(date: str, rating: int, time_control_bucket: str)`
       - `RatingHistoryResponse(chess_com: list[RatingDataPoint], lichess: list[RatingDataPoint])`
       - `WDLByCategory(label: str, wins: int, draws: int, losses: int, total: int, win_pct: float, draw_pct: float, loss_pct: float)`
       - `GlobalStatsResponse(by_time_control: list[WDLByCategory], by_color: list[WDLByCategory])`

    2. Create `app/repositories/stats_repository.py`:
       - `query_rating_history(session, user_id, platform, recency_cutoff)` — SELECT played_at (cast to date in UTC via `func.timezone("UTC", Game.played_at)`), user_rating, time_control_bucket WHERE platform=X, user_rating IS NOT NULL, played_at IS NOT NULL, ORDER BY played_at. Apply recency_cutoff if provided.
       - `query_results_by_time_control(session, user_id, recency_cutoff)` — SELECT time_control_bucket, result, user_color WHERE time_control_bucket IS NOT NULL. Apply recency_cutoff if provided.
       - `query_results_by_color(session, user_id, recency_cutoff)` — SELECT user_color, result WHERE user_color IS NOT NULL. Apply recency_cutoff if provided.

    3. Create `app/services/stats_service.py`:
       - `get_rating_history(session, user_id, recency)` — call recency_cutoff(), query both platforms, map rows to RatingDataPoint list per platform, return RatingHistoryResponse.
       - `get_global_stats(session, user_id, recency)` — call recency_cutoff(), query results by TC and by color. Aggregate using derive_user_result(). Compute WDL percentages. Return GlobalStatsResponse. For by_time_control, labels are "Bullet", "Blitz", "Rapid", "Classical" (title-cased from bucket). For by_color, labels are "White", "Black".

    4. Create `app/routers/stats.py`:
       - `router = APIRouter(tags=["stats"])`
       - `GET /stats/rating-history` with `recency: str | None = Query(default=None)` — calls stats_service.get_rating_history
       - `GET /stats/global` with `recency: str | None = Query(default=None)` — calls stats_service.get_global_stats
       - Both require `Depends(current_active_user)` and `Depends(get_async_session)`

    5. Register router in `app/main.py` — add `from app.routers.stats import router as stats_router` and `app.include_router(stats_router)`.

    IMPORTANT: Reuse `recency_cutoff()` and `derive_user_result()` from `app/services/analysis_service.py` — do NOT duplicate. Import them.
    IMPORTANT: Use `func.timezone("UTC", Game.played_at)` before any date operations per STATE.md decision on UTC normalization.
  </action>
