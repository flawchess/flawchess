why_human: The 2026-07-12 plan-07 sign-off explicitly certified "bot never flags" — the model D-15/D-16/D-18 reversed the next day. The human gate certified behavior that no longer exists. The deadline band constants (BOT_MOVES_TO_GO, BOT_THINK_INCREMENT_SHARE, the [800ms, 15s] clamp) are explicitly "tuned by feel"; only a human playing can judge the pacing.

### 2. Hidden-tab behavior in a real browser
expected: Background the tab ~30s during the bot's think, then again during your own turn. Neither clock is drained by the hidden interval; no side flags; clocks resume from where they paused.
why_human: Real background-tab interval throttling and real backgrounded Web Worker execution cannot be reproduced in jsdom — the tests simulate visibilitychange, they do not exercise the actual throttled-interval + backgrounded-worker interaction.
note: Also worth trying the newly-fixed path — open /bots in a BACKGROUND tab (middle-click the link), leave it 30s, then switch to it. The game must not already be lost on time. This was a real bug found and fixed during this run (commit 21bdd932).
