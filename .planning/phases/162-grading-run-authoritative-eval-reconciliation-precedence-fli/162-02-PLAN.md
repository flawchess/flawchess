Wire the flipped precedence into the Analysis page's union and label layer: extend `unionSans` with the free run's own top-2 (D-02) gated on a bestmove-commit signal (D-09), compute the single canonical `reconciledBestUci` once (D-11 "resolve once, thread everywhere"), and re-source the chart's Best/Good labels from that reconciled argmax instead of the free-run `bestSan` pin (D-03).

The `reconciledBestUci` memo created here is the ONE source of truth that Wave 3 threads into the arrow, verdict, eval bar, and card.
Output: A gated union extension, a canonical reconciled-best memo, a re-sourced `qualityBySan` pin, and integration tests for the gate and the mirror-image label case.
</objective>
