and runs under the new dispatch key. Getting these right propagates to fork, hanging-piece,
skewer, pin, interference, and trapped-piece simultaneously.

user input, no auth/session/access-control changes, no network calls. The detector reads only
already-stored PV/eval data (`game_positions.pv`, `game_positions.eval_mate`). There is no new
trust boundary crossed by this plan.

| Boundary | Description |
|----------|-------------|
| stored PV/eval → detector | already-validated data read at classify time; no external/untrusted ingress in this plan |
