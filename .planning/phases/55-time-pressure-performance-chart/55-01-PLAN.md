```python
def _build_bucket_series(
    buckets: list[list[float]],
) -> list[TimePressureBucketPoint]:
    """Build 10-point series from accumulated [score_sum, game_count] pairs."""
    points: list[TimePressureBucketPoint] = []
    for i, (score_sum, count) in enumerate(buckets):
        lo = i * BUCKET_WIDTH_PCT
        hi = (i + 1) * BUCKET_WIDTH_PCT
        label = f"{lo}-{hi}%"
        score = (score_sum / count) if count > 0 else None
        points.append(TimePressureBucketPoint(
            bucket_index=i,
            bucket_label=label,
            score=score,
            game_count=int(count),
        ))
    return points
```

Add `_compute_time_pressure_chart` function following the EXACT same loop structure as `_compute_clock_pressure` (same row unpacking, same guards):
```python
def _compute_time_pressure_chart(
    clock_rows: Sequence[Row[Any] | tuple[Any, ...]],
) -> TimePressureChartResponse:
    """Compute time-pressure performance chart data from clock_rows.

    Reuses the same clock_rows already fetched by query_clock_stats_rows in
    get_endgame_overview -- no additional DB query needed.

    For each game with both clocks and valid time_control_seconds:
    - Bucket user's time% -> accumulate user_score into user_series
    - Bucket opp's time% -> accumulate (1 - user_score) into opp_series

    Returns rows for time controls with >= MIN_GAMES_FOR_CLOCK_STATS games.
    """
```

The loop iterates over clock_rows with the same unpacking as _compute_clock_pressure:
- game_id = row[0], time_control_bucket = row[1], time_control_seconds = row[2],
  termination = row[3], result = row[4], user_color = row[5],
  ply_array = row[6], clock_array = row[7]
- Skip if time_control_bucket is None
- Extract clocks via _extract_entry_clocks(ply_array, clock_array, user_color)
- Skip if user_clock is None or opp_clock is None
- Skip if time_control_seconds is None or time_control_seconds <= 0
- Compute user_pct = user_clock / time_control_seconds * 100
- Compute opp_pct = opp_clock / time_control_seconds * 100
- Clamp buckets: user_bucket = min(int(user_pct / BUCKET_WIDTH_PCT), NUM_BUCKETS - 1)
- Derive user_score: {"win": 1.0, "draw": 0.5, "loss": 0.0}[derive_user_result(result, user_color)]
- Accumulate: tc_user_buckets[tc][user_bucket][0] += user_score, [1] += 1
- Accumulate: tc_opp_buckets[tc][opp_bucket][0] += (1.0 - user_score), [1] += 1
- Track total games per tc via tc_game_count[tc] (count games with both clocks)

Use two defaultdict accumulators:
```python
tc_user_buckets: dict[str, list[list[float]]] = defaultdict(
    lambda: [[0.0, 0] for _ in range(NUM_BUCKETS)]
)
tc_opp_buckets: dict[str, list[list[float]]] = defaultdict(
    lambda: [[0.0, 0] for _ in range(NUM_BUCKETS)]
)
tc_game_count: dict[str, int] = defaultdict(int)
```

After the loop, build rows in _TIME_CONTROL_ORDER, filtering out tc's with tc_game_count[tc] < MIN_GAMES_FOR_CLOCK_STATS. Use cast(Literal["bullet", "blitz", "rapid", "classical"], tc) for the time_control field (same pattern as _compute_clock_pressure).
