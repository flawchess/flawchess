<objective>
Build the single shared Python compute path that ports the four locked lichess formulas.

This plan creates `app/services/accuracy_acpl.py` — pure stdlib `math` (reusing
`LICHESS_K` from `eval_utils.py`, no re-declared constant) — with one public
orchestrator `compute_game_accuracy_acpl(positions, *, start_color="white")` that maps
the stored White-POV, post-move-shifted `game_positions` evals to before/after mover-POV
pairs, applies the win% (D-08), per-move accuracy (D-09), windowed game accuracy (D-10),
and ACPL (D-11) formulas, and returns per-color computed accuracy/ACPL — or `None` when
the per-ply eval sequence has an interior hole. It also creates the hand-checked fixture
+ edge-case unit tests (D-07), including the game 296343 ACPL reproduction that
simultaneously validates sign + post-move shift + mover parity + the ACPL formula.

Purpose: This one function is called by BOTH the live hook (Plan 03) and the backfill
(Plan 04) — the single-path guarantee that keeps formula logic un-duplicated.

Output: `app/services/accuracy_acpl.py` + `tests/services/test_accuracy_acpl.py`.
</objective>
