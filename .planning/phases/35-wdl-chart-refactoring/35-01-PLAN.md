    ```typescript
    import type { ReactNode } from 'react';
    import type { WDLRowData } from '@/types/charts';

    interface WDLChartRowProps {
      /** WDL statistics data */
      data: WDLRowData;

      /** Optional row label (left side of header) */
      label?: string;

      /** Optional info popover rendered after the label */
      infoPopover?: ReactNode;

      /** Optional games link — renders as a React Router Link with ExternalLink icon */
      gamesLink?: string;
      /** Optional click handler for the games link (e.g. for side effects before navigation) */
      onGamesLinkClick?: () => void;
      /** data-testid for the games link element */
      gamesLinkTestId?: string;
      /** aria-label for the games link */
      gamesLinkAriaLabel?: string;

      /** When present, renders a grey-outlined proportional game count bar. Value = max total across all rows for proportional sizing. */
      maxTotal?: number;

      /** Minimum games threshold for reliable stats. Below this, bar and legend are dimmed. Defaults to MIN_GAMES_FOR_RELIABLE_STATS (10). */
      minGamesForReliable?: number;

      /** Bar height class. Defaults to 'h-5' (reference implementation). */
      barHeight?: 'h-5' | 'h-6';

      /** Show "(low)" warning next to game count when below minGamesForReliable. Default false. */
      showLowWarning?: boolean;

      /** data-testid for the row container */
      testId?: string;
    }
    ```

    Implementation must replicate the EndgameWDLChart row pattern exactly:

    a. **Header row** (only if `label` is provided): flex row with label + optional infoPopover on left, game count + optional games link on right. Game count shows `{data.total} games`. If `showLowWarning` is true and `data.total < minGamesForReliable`, append `<span className="text-amber-500 ml-1" title="Small sample size — percentages may be unreliable">(low)</span>`. Games link uses `<Link to={gamesLink}>` with `<ExternalLink className="h-3.5 w-3.5" />`.

    b. **Stacked WDL bar**: `<div className={cn('flex w-full overflow-hidden rounded mb-0', barHeight)}>` with opacity dimming when `data.total < minGamesForReliable` via `style={{ opacity: UNRELIABLE_OPACITY }}`. Three inner divs for win/draw/loss (only rendered when pct > 0), each with `className="transition-all"` and `style={{ width: \`${data.X_pct}%\`, backgroundColor: WDL_X, backgroundImage: GLASS_OVERLAY }}`.

    c. **Game count bar** (only if `maxTotal` is provided): `<div className="h-2 mt-0.5 mb-1"><div className="h-full rounded-sm" style={{ width: \`${(data.total / maxTotal) * 100}%\`, border: '1px solid oklch(0.6 0 0)', backgroundColor: 'transparent' }} /></div>`

    d. **WDL legend text row**: `<div className="flex justify-center gap-3 text-sm" style={dimStyle}>` with three spans: `W: {data.wins} ({Math.round(data.win_pct)}%)`, `D: {data.draws} ({Math.round(data.draw_pct)}%)`, `L: {data.losses} ({Math.round(data.loss_pct)}%)` — each colored with `WDL_WIN`, `WDL_DRAW`, `WDL_LOSS` respectively. Apply same opacity dimming as the bar.

    e. **Empty state**: If `data.total === 0`, render `<div className="space-y-2"><div className={cn('w-full rounded bg-muted', barHeight)} /><p className="text-center text-sm text-muted-foreground">No games matched</p></div>`.

    Imports: `Link` from `react-router-dom`, `ExternalLink` from `lucide-react`, `cn` from `@/lib/utils`, `WDL_WIN`, `WDL_DRAW`, `WDL_LOSS`, `GLASS_OVERLAY`, `MIN_GAMES_FOR_RELIABLE_STATS`, `UNRELIABLE_OPACITY` from `@/lib/theme`, `WDLRowData` from `@/types/charts`.
