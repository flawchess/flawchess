**1. Add `_extract_entry_clocks` helper to `app/services/endgame_service.py`:**

Place near other private helpers (after `_compute_score_gap_material`):

```python
def _extract_entry_clocks(
    plies: list[int],
    clocks: list[float | None],
    user_color: str,
) -> tuple[float | None, float | None]:
    """Return (user_entry_clock, opp_entry_clock) at endgame entry.

    user_color determines ply parity: white=even, black=odd.
    Returns (None, None) if the expected entry plies have no clock data.
    """
    user_parity = 0 if user_color == "white" else 1
    user_clock: float | None = None
    opp_clock: float | None = None
    for ply, clock in zip(plies, clocks):
        if ply % 2 == user_parity and user_clock is None:
            user_clock = clock
        elif ply % 2 != user_parity and opp_clock is None:
            opp_clock = clock
        if user_clock is not None and opp_clock is not None:
            break
    return user_clock, opp_clock
```

class _ClockAccumulator(TypedDict):
    total_endgame_game_ids: set[int]
    timeout_win_game_ids: set[int]
    timeout_loss_game_ids: set[int]
    clock_user_values: list[float]
    clock_opp_values: list[float]

`_compute_clock_pressure` takes `clock_rows: Sequence[Row[Any] | tuple[Any, ...]]` and returns `ClockPressureResponse`.

Algorithm:
1. Initialize accumulators per time_control_bucket using `defaultdict`.
2. For each row: extract game_id (idx 0), time_control_bucket (idx 1), time_control_seconds (idx 2), termination (idx 3), result (idx 4), user_color (idx 5), ply_array (idx 6), clock_array (idx 7).
3. Skip rows where `time_control_bucket is None`.
4. Add game_id to `total_endgame_game_ids` set.
5. Call `_extract_entry_clocks(ply_array, clock_array, user_color)`.
6. If BOTH user_clock and opp_clock are not None:
   - Append user_clock to clock_user_values, opp_clock to clock_opp_values
   - Append (user_clock - opp_clock) to clock_diff_values
   - If time_control_seconds is not None and time_control_seconds > 0:
     - Append (user_clock / time_control_seconds * 100) to clock_pct_user_values
     - Append (opp_clock / time_control_seconds * 100) to clock_pct_opp_values
7. For timeout: determine user outcome via `derive_user_result(result, user_color)`. If termination == "timeout": add game_id to timeout_win_game_ids (if win) or timeout_loss_game_ids (if loss).
8. After processing all rows, build ClockStatsRow for each bucket in fixed order: ["bullet", "blitz", "rapid", "classical"].
9. Use `_TIME_CONTROL_LABELS` dict: `{"bullet": "Bullet", "blitz": "Blitz", "rapid": "Rapid", "classical": "Classical"}`.
10. Define `MIN_GAMES_FOR_CLOCK_STATS = 10` as a module-level constant.
11. Filter out rows where `len(total_endgame_game_ids) < MIN_GAMES_FOR_CLOCK_STATS`.
12. For net_timeout_rate: `(len(timeout_win_game_ids) - len(timeout_loss_game_ids)) / len(total_endgame_game_ids) * 100`.
13. For averages: use `statistics.mean()` or manual sum/len. Use `mean(values) if values else None`.
