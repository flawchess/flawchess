<objective>
Build `useFlawChessEngine` — the React hook that drives the frozen `SearchRunner` (`mctsSearch`)
against the real Phase 154 providers (`createWorkerPool()` + `createMaiaQueue()`) and exposes a
live-refining ranked-lines snapshot to the page. This is the anytime-emit engine behind
DISPLAY-01: lines appear immediately from the first `onSnapshot` and refine at a 150ms throttle.

Purpose: The hook owns provider lifetime, the FEN-navigation trigger, the AbortController +
`pool.stopAll()` cleanup, and the onSnapshot throttle. It is the single integration point the
page (Plan 04) mounts.

Output: `useFlawChessEngine.ts` returning a flat data snapshot, plus its filled-in test file.
</objective>
