---

## Sampling Rate

- **After every task commit:** Run the scoped command (the `<verify><automated>` line on that task).
- **After every plan completes:** Run the plan's terminal Pre-PR checklist task (or, for plans without one, the full quick-run command for the layer touched).
- **Before `/gsd-verify-work`:** Full suite must be green (backend `uv run pytest -x` + frontend `npm test -- --run` + ty + lint + knip).
- **Max feedback latency:** ~120 seconds (scoped) / ~3 minutes (full).
