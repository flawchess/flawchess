       - **Same game contributes to two cells**: the original query bucketed each game once by `user_quintile` and computed per-user mean score. The rerun must produce TWO per-game contributions: one to `(tc, user_quintile)` with `user_score = derive_user_result(result, user_color)` mapped to 1/0.5/0, and one to `(tc, opp_quintile)` with `opp_score = inverted_outcome` mapped to 1/0.5/0.
       - **Per-user aggregate**: for each user, compute per-(TC, quintile) `user_mean_score` (the user's own quintile bucket) AND per-(TC, quintile) `opp_mean_score` (the opponent's quintile bucket from the user's same games). Then per-user delta = user_mean_score_in_Q − opp_mean_score_in_Q.
       - **Per-(TC, quintile) p25/p75**: take p25 and p75 of those per-user deltas across users (ELO pooled, sparse-cell exclusion as before).

       Use the `mcp__flawchess-benchmark-db__query` MCP tool. Sketch the SQL:
       ```sql
       WITH per_game AS (
         -- for each user's game, compute user_quintile, opp_quintile, user_score, opp_score
         -- (mirroring the new _iterate_clock_rows logic from Plan 88-09)
       ),
       per_user_per_cell AS (
         SELECT user_id, tc, quintile,
                AVG(score) FILTER (WHERE side='user') AS user_mean_score,
                AVG(score) FILTER (WHERE side='opp')  AS opp_mean_score,
                COUNT(*)   FILTER (WHERE side='user') AS n_user,
                COUNT(*)   FILTER (WHERE side='opp')  AS n_opp
         FROM (
           SELECT user_id, tc, user_quintile AS quintile, user_score AS score, 'user' AS side FROM per_game
           UNION ALL
           SELECT user_id, tc, opp_quintile  AS quintile, opp_score  AS score, 'opp'  AS side FROM per_game
         ) s
         GROUP BY user_id, tc, quintile
         HAVING COUNT(*) FILTER (WHERE side='user') >= MIN_GAMES_PER_PRESSURE_BIN
            AND COUNT(*) FILTER (WHERE side='opp')  >= MIN_GAMES_PER_PRESSURE_BIN
       ),
       per_user_delta AS (
         SELECT user_id, tc, quintile, user_mean_score - opp_mean_score AS delta
         FROM per_user_per_cell
       )
       SELECT tc, quintile,
              percentile_cont(0.25) WITHIN GROUP (ORDER BY delta) AS p25,
              percentile_cont(0.50) WITHIN GROUP (ORDER BY delta) AS p50,
              percentile_cont(0.75) WITHIN GROUP (ORDER BY delta) AS p75,
              COUNT(*) AS n_users
         FROM per_user_delta
        WHERE NOT (...)  -- sparse-cell exclusion
        GROUP BY tc, quintile;
       ```
