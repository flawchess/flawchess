Mirror that function's docstring conventions, n-gate behavior, and import structure. Compute `user_score = (user_w + 0.5 * user_d) / user_n`. Guard `user_n == 0` by returning `(0.0, None, None, None)`. For `user_n == 1`, return `(user_score - cohort_score, None, None, None)` (no CI from a single observation). For `user_n >= 2`, compute Wilson CI via `wilson_bounds(user_score, user_n)` and transplant: `ci_low = wilson_lo - cohort_score`, `ci_high = wilson_hi - cohort_score`. Compute p_value via `_wilson_score_test_vs_ref(user_score, user_n, cohort_score)`. Apply the n-gate `p_out = p_value if user_n >= CONFIDENCE_MIN_N else None` (same gate as `compute_paired_difference_test`). Return `(delta, p_out, ci_low, ci_high)`.
