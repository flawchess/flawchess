## Decisions Made

- Equal-weighted Skill mean (D-01): active bucket = n >= CONFIDENCE_MIN_N=10; dropped buckets not counted in denominator. This means with 2 active buckets, Skill = (m_c + m_r) / 2, not (m_c + 0 + m_r) / 3.
- CI propagation via variance-of-sum/n_active² (Open Q §3 Option A): assumes independence between the 3 paired-z outputs. SE_skill = sqrt(sum(SE_b²)) / n_active. Conservative but tractable.
- dict[str, list[float]] throughout (not dict[MaterialBucket, list[float]]): Python dict is invariant — assigning a defaultdict[str, list[float]] to dict[Literal["conversion","parity","recovery"], list[float]] triggers a ty error. Using str throughout avoids this with no loss of runtime safety.
- Cohort-empty guard: when bucket n=0, compute_paired_difference_test returns mean=0.0 (its contract). We gate to None before putting on the wire to avoid false "user scored 0.0" signals.
