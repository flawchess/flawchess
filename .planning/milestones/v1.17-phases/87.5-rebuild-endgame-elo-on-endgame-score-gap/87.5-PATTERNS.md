**New pure mapping helper** (replaces `_conversion_elo_from_skill`, lines 1320–1342):
```python
def _endgame_elo_from_score_gap(actual_elo: float, eg_score_gap: float, k: float) -> int:
    """Additive ELO offset from Endgame Score Gap (Phase 87.5).

    endgame_elo = round(actual_elo + K * eg_score_gap)

    At eg_score_gap = 0, endgame_elo == actual_elo exactly.
    Pure function. Floor/ceiling clamping (if needed) mirrors existing
    endgame_elo_timeline contract — do not introduce a new clamp policy.
    """
    return round(actual_elo + k * eg_score_gap)
```

**Weekly-series producer rewire** (replaces lines 1506–1542 in `_compute_endgame_elo_weekly_series`):

**Analog:** `tests/services/test_conversion_elo_recenter.py` (Phase 87.4 — full file).

**Structure to mirror:**
```python
class TestEndgameEloFromScoreGap:
    def test_zero_gap_preserves_actual_elo(self) -> None:
        # eg_score_gap=0 => endgame_elo == actual_elo at any rating
        assert _endgame_elo_from_score_gap(1500.0, 0.0, K) == 1500

    def test_positive_gap_lifts_elo(self) -> None:
        # p95 gap ~ +0.22; K=450 => +99 ELO above actual
        result = _endgame_elo_from_score_gap(1500.0, 0.22, K)
        assert result == round(1500.0 + K * 0.22)

    def test_negative_gap_drags_elo(self) -> None:
        # p05 gap ~ -0.23; K=450 => ~-104 ELO below actual
        result = _endgame_elo_from_score_gap(1500.0, -0.23, K)
        assert result == round(1500.0 + K * -0.23)

    def test_rounding_contract(self) -> None:
        # Result is int (not float)
        assert isinstance(_endgame_elo_from_score_gap(1500.0, 0.01, K), int)
```

**What to adjust vs analog:** swap `TestAffineRecenter` / `TestConversionEloInvariant` classes for the above. There is no PIVOT/ALPHA to test — the formula is just `round(actual_elo + K * gap)`. The invariant test is `gap=0 => result==actual_elo` (exact, no approx needed because the float multiplication is exact at 0). Add a clamping test only if the implementation adds a clamp.
