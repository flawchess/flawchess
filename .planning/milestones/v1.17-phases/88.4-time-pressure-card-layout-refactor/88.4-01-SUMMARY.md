# Phase 88.4 Plan 01: ScoreGapByTimePressureChart Component Summary

**One-liner:** Recharts ComposedChart line chart over 4 ordinal pressure-bucket labels with TC-collapsed ±0.06 zone bands, ErrorBar CI whiskers, zone-colored dots, and a hover tooltip replacing the four stacked QuintileRow bullet charts.

## What Was Built

A new self-contained `ScoreGapByTimePressureChart` component (SC-3 contract) that:

- Accepts `quintiles: PressureQuintileBullet[]` and `tc: 'bullet' | 'blitz' | 'rapid' | 'classical'`
- Filters Q4 (`quintile_index > MAX_VISIBLE_QUINTILE_INDEX=3`) and zero-n bins from the data array
- Renders a Recharts `ComposedChart` with:
  - Three flat `ReferenceArea` zone bands using the TC-collapsed neutral zone `[-0.06, +0.06]`
  - Category `XAxis` with 4 ordinal labels ("0-20% Time" through "60-80% Time")
  - Fixed Y domain `[-0.30, +0.30]` matching `PRESSURE_DELTA_DOMAIN`
  - White `Line` with `connectNulls={false}` (line breaks at omitted zero-n bins)
  - Zone-colored custom dot render via `pressureDeltaZoneColor`
  - `ErrorBar` child of `Line` with `dataKey="ciError"` (pre-computed offsets; undefined suppresses whisker for null CI bins)
  - `ChartTooltip` with `data-testid="score-gap-tooltip"` showing signed delta %, opp score %, and n
- Exports `PRESSURE_SCORE_GAP_NEUTRAL_MIN = -0.06` and `PRESSURE_SCORE_GAP_NEUTRAL_MAX = 0.06` with benchmark citation
- Root div has `data-testid="score-gap-by-time-pressure-chart-{tc}"`, `role="img"`, `aria-label`

The Wave 0 test scaffold was written first (9 tests, RED), then the component was implemented to turn all tests GREEN.
