# Phase 88.4 Plan 03: EndgameTimePressureCard Refactor (SC-2 + SC-3) Summary

**One-liner:** Replaced ClockGapRow label + ThreeStatRow with a 3-col header row (You/Gap+info/Opp) above the Clock Gap bullet (SC-2), and replaced four stacked QuintileRow/EmptyBinRow bullet charts with a single ScoreGapByTimePressureChart (SC-3), with knip and tsc clean throughout.

## What Was Built

### SC-2: ClockGapHeaderRow

A new `ClockGapHeaderRow({ gap, card })` sub-component replaces the old `ClockGapRow` label span and the `You:`/`Opponents:` items from `ThreeStatRow`. It renders a `grid grid-cols-3 items-center text-sm tabular-nums mb-2` row above the Clock Gap `MiniBulletChart`:

- Left cell (`data-testid=…-my-avg-time`): "You: X% (Ns)" using `formatPctSecs`
- Center cell: "Gap:" label + signed value (`data-testid=…-clock-gap-value`) + migrated `MetricStatPopover` (`testId=…-clock-gap-info`)
- Right cell (`data-testid=…-opp-avg-time`): "Opp: X% (Ns)" using `formatPctSecs`
- Font-tinting triple-gate preserved from `ClockGapRow` (n >= MIN, isConfident, outside neutral band)

### SC-2: NetFlagRateRow

A new `NetFlagRateRow({ card })` sub-component holds only the surviving "Net flag rate:" stat. The old `ThreeStatRow` is deleted. `tintForNetTimeoutRate`, `formatNetTimeoutRate`, and `InfoPopover` are all kept intact.

### SC-3: ScoreGapByTimePressureChart Integration

`QuintileRow` and `EmptyBinRow` sub-components deleted. The quintile loop in the "Score Gap by Remaining Time" section is replaced with:

```tsx
<div data-testid={`time-pressure-card-${card.tc}-score-gap-chart`}>
  <ScoreGapByTimePressureChart quintiles={card.quintiles} tc={card.tc} />
</div>
```

Orphaned imports removed: `getPressureBinBand`, `PRESSURE_DELTA_DOMAIN`, `PRESSURE_DELTA_CENTER`, `UNRELIABLE_OPACITY`, `pressureDeltaZoneColor`. `clampDeltaCi` kept (still used for Clock Gap bullet CI clamping). `isConfident` and `Swords` kept.

### Test Updates

Deleted 6 obsolete describe blocks (n=0 bin slot, sparse bins, triple-gate font coloring, Phase 88.1 opp-quintile rename, Plan 88-13 A-4 Q4 hidden, quintile bucket labels). Added SC-2 describe (3 cases) and SC-3 describe (2 cases). Updated Plan 88-14 A-3 describe to reference the new `ClockGapHeaderRow` testids.
