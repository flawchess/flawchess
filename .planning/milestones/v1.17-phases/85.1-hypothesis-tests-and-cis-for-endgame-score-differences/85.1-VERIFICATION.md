# Phase 85.1: Hypothesis tests + CIs for Endgame Score Differences — Verification Report

**Phase Goal:** Add two-sided p-values and 95% CIs to the two derived metrics in the Phase 85 "Endgame Score Differences" card. Endgame Score Gap (Endgame Score − Non-Endgame Score) is an independent two-sample z-test on the chess-score difference, using the trinomial empirical variance per side. Achievable Score Gap (Endgame Score − Achievable Score) is a paired one-sample z-test on per-game differences over the same `bucket_rows` cursor that already feeds `entry_expected_score`. Surface CIs as bullet whiskers on each `ScoreGapRow`.
