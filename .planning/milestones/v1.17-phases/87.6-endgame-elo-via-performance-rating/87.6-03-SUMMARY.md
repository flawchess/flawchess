## What Was Built

### TimePoint Schema Extension (app/schemas/insights.py)
Added two optional fields to `TimePoint`:
- `endgame_elo: int | None = None` — FIDE Performance Rating from endgame game subset
- `non_endgame_elo: int | None = None` — FIDE Performance Rating from non-endgame game subset

Both fields are gated: populated ONLY for `endgame_elo_timeline` series, None elsewhere.

### Service Layer: 6-Tuple Threading (app/services/insights_service.py)
- `_series_for_endgame_elo_combo`: 4-tuple extended to 6-tuple `(date, gap, n, actual_elo, endgame_elo, non_endgame_elo)`
- `_weekly_points_to_time_points_with_elo`: updated signature and body to consume 6-tuple, emit all three PR fields, compute weighted means for `endgame_elo` and `non_endgame_elo` in the `all_time` aggregation path

### LLM Layer: THREE Summary Blocks (app/services/insights_llm.py)
- `_PROMPT_VERSION` bumped from `endgame_v34` with prepended v34 changelog paragraph documenting: K=450 deleted, FIDE PR formula, midpoint property, `non_endgame_elo` field, THREE blocks, sentence skeleton
- New `_non_endgame_elo_per_bucket(points)` helper: reads `pt.non_endgame_elo` directly (analogous to `_endgame_elo_per_bucket` which derives from `pt.actual_elo + pt.value`)
- New `_render_non_endgame_elo_summary_block(...)` helper: emits `[summary non_endgame_elo | dim]` with same window/std/trend/shift structure as the endgame_elo block
- Subsection dispatcher: Phase 87.6 comment + second `lines.extend(_render_non_endgame_elo_summary_block(...))` call immediately after the existing endgame_elo block
- `_render_series_block`: extended `emit_elo` branch to include `non_eg_elo=<int>` when `pt.non_endgame_elo is not None`

### Prompt File Rewrite (app/prompts/endgame_insights.md)
- UI-label table: added `non_endgame_elo` row
- Series row doc: updated from two-number to three-number format (`gap=, elo=, non_eg_elo=`) with midpoint property note
- TWO → THREE summary blocks section rewritten: updated descriptions for all three blocks, new pairing rule with sentence skeleton
- Glossary: `endgame_elo` entry rewritten under FIDE PR framing; new `non_endgame_elo` entry inserted after it

### Ancillary Changes
- CHANGELOG.md: Phase 87.6 Changed bullet prepended above Phase 87.5 entry
- Design note: Phase 87.6 supersession header prepended to `endgame-elo-rebuild-on-score-gap.md`
