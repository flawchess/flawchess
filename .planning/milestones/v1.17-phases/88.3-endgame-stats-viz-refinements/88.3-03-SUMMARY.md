## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] ConnectorArrows selector false positive in test**
- **Found during:** Task 2 (first test run)
- **Issue:** The plan-suggested selector `[class*="absolute"][class*="inset-0"]` matched a `div.absolute.inset-0` rendered by the `charcoal-texture` CSS pseudo-element backdrop (a gradient overlay), not the ConnectorArrows component. Test failed with an unexpected match.
- **Fix:** Changed selector to `[aria-hidden="true"][class*="pointer-events-none"]`, which is unique to the ConnectorArrows overlay wrapper (it had `class="absolute inset-0 hidden lg:block pointer-events-none" aria-hidden="true"`). The charcoal-texture backdrop does not carry `pointer-events-none`.
- **Files modified:** EndgameOverallPerformanceSection.test.tsx
- **Commit:** d7a3d384
