# Phase 88.3 Plan 04: SC-4 Palmtree Glyph + 6-Chart Inactivity Gap Rollout Summary

SC-4 gap-closure: replaced the compact fontSize-11 SVG text break label with a lucide Palmtree glyph at BREAK_LABEL_FONT_SIZE (14px, text-sm floor), extracted the duplicated inline ReferenceLine map into one shared render helper, and rolled it out to all 6 ordinal-axis timeline charts.

## What Was Built

**Task 1 — Shared helper + theme constant + tests:**

Added `BREAK_LABEL_FONT_SIZE = 14` to `theme.ts`. Created `InactivityGapReferenceLines.tsx` as a plain function returning `ReactElement[]`. The function calls `computeInactivityGaps` (no duplication), maps each gap to a `<ReferenceLine>` with a Palmtree glyph + fontSize-14 label, and is direct-child safe for Recharts chart internals. `data-testid="inactivity-gap-label"` on the label group and `data-testid="inactivity-gap-glyph"` on the Palmtree enable deterministic test assertions. Wrote 12 tests covering pure array returns and render cases.

**Task 2 — Refactor two endgame charts:**

Removed `computeInactivityGaps` import, `inactivityGaps` useMemo, and the inline `inactivityGaps.map(...)` block from both `EndgameScoreOverTimeChart.tsx` and `EndgameEloTimelineSection.tsx`. Replaced with `inactivityGapReferenceLines({ dates, yAxisId: 'value' })` and `inactivityGapReferenceLines({ dates: allDates, yAxisId: 'elo' })` respectively. Updated both test files to target `data-testid` anchors instead of raw `.recharts-reference-line-line` class, added glyph presence assertion. SC-1/SC-2/SC-3 behavior and tests unchanged (23 tests pass).

**Task 3 — Wire 4 remaining charts:**

- `ScoreChart`: helper call with no yAxisId (single default axis), placed before the bookmark Line series
- `EndgameClockDiffOverTimeChart`: helper call with `yAxisId: 'value'` alongside the existing horizontal `y={0}` baseline ReferenceLine (independent axes, no overlap)
- `RatingChart`: defensive `sortedGapDates = useMemo(() => [...allDates].sort(), [allDates])` to guarantee ascending-sort invariant; helper call with no yAxisId; covers both Chess.com and Lichess instances

Created `ScoreChart.test.tsx` (4 cases), `RatingChart.test.tsx` (5 cases including out-of-order-dates defensive-sort proof), extended `EndgameClockDiffOverTimeChart.test.tsx` (4 gap-annotation cases + baseline coexistence regression).
