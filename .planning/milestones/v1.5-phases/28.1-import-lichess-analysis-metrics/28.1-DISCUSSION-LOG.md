## Chess.com API Verification

User requested verification of whether chess.com provides inaccuracy/mistake/blunder counts or ACPL before proceeding.

**Research performed:**
- Checked chess.com forum (moderator response on "Insight data in public APIs")
- Checked chess.com Published-Data API documentation
- Confirmed: API provides only `accuracies.white`/`accuracies.black` — no ACPL, no inaccuracy/mistake/blunder counts
- API is in "bug fix mode only" with no new features planned

**Conclusion:** These metrics are lichess-only. User confirmed to proceed with lichess-only import.
