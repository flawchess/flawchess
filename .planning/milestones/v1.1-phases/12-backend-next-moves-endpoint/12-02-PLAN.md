**Implementation steps:**

1. Compute recency cutoff: `cutoff = recency_cutoff(request.recency)`

2. Compute position_stats by reusing `query_all_results` with `hash_column=GamePosition.full_hash` and `target_hash=request.target_hash`:
   ```python
   from app.models.game_position import GamePosition
   all_rows = await query_all_results(
       session=session, user_id=user_id,
       hash_column=GamePosition.full_hash,
       target_hash=request.target_hash,
       time_control=request.time_control, platform=request.platform,
       rated=request.rated, opponent_type=request.opponent_type,
       recency_cutoff=cutoff, color=request.color,
   )
   ```
   Then compute W/D/L same as `analyze()` does (loop over rows, derive_user_result, count):
   ```python
   wins = draws = losses = 0
   for result, user_color in all_rows:
       outcome = derive_user_result(result, user_color)
       if outcome == "win": wins += 1
       elif outcome == "draw": draws += 1
       else: losses += 1
   total = wins + draws + losses
   win_pct = round(wins / total * 100, 1) if total > 0 else 0.0
   draw_pct = round(draws / total * 100, 1) if total > 0 else 0.0
   loss_pct = round(losses / total * 100, 1) if total > 0 else 0.0
   position_stats = WDLStats(wins=wins, draws=draws, losses=losses, total=total,
       win_pct=win_pct, draw_pct=draw_pct, loss_pct=loss_pct)
   ```

3. Query next moves:
   ```python
   move_rows = await query_next_moves(
       session=session, user_id=user_id, target_hash=request.target_hash,
       time_control=request.time_control, platform=request.platform,
       rated=request.rated, opponent_type=request.opponent_type,
       recency_cutoff=cutoff, color=request.color,
   )
   ```

4. If no moves found, return early:
   ```python
   if not move_rows:
       return NextMovesResponse(position_stats=position_stats, moves=[])
   ```

5. Collect result_hashes and query transposition counts:
   ```python
   result_hashes = list({row.result_hash for row in move_rows})
   trans_counts = await query_transposition_counts(
       session=session, user_id=user_id, result_hash_list=result_hashes,
       time_control=request.time_control, platform=request.platform,
       rated=request.rated, opponent_type=request.opponent_type,
       recency_cutoff=cutoff, color=request.color,
   )
   ```

6. Compute result_fen for each unique result_hash. Use a helper function `_compute_result_fens`:
   ```python
   def _compute_result_fens(move_rows: list, target_hash: int, session_games_cache: dict) -> dict[int, str]:
       """Compute board_fen for each result_hash by replaying the move on a known position.

       Since we know the source position's full_hash corresponds to a specific board state,
       and we have the move_san, we can find the result FEN by:
       1. Getting one sample (game_id, ply) per result_hash from game_positions
       2. Fetching that game's PGN and replaying to ply+1
       """
   ```

   **However**, the simplest correct approach for result_fen:
   - For each result_hash, query `game_positions` for one sample row: `SELECT game_id, ply FROM game_positions WHERE user_id=? AND full_hash=? LIMIT 1`
   - Then fetch that game's PGN from the `games` table
   - Replay with python-chess: iterate `chess.pgn.read_game()`, push moves up to the target ply, call `board.board_fen()`

   Create an async helper:
   ```python
   async def _fetch_result_fens(
       session: AsyncSession,
       user_id: int,
       result_hashes: list[int],
   ) -> dict[int, str]:
   ```

   Implementation:
   - For each result_hash in the list, run a single batch query using `DISTINCT ON (full_hash)`:
     ```python
     from sqlalchemy import select, text
     stmt = (
         select(GamePosition.full_hash, GamePosition.game_id, GamePosition.ply)
         .where(
             GamePosition.user_id == user_id,
             GamePosition.full_hash.in_(result_hashes),
         )
         .distinct(GamePosition.full_hash)
         .order_by(GamePosition.full_hash, GamePosition.game_id)
     )
     samples = await session.execute(stmt)
     sample_rows = list(samples.all())
     ```
   - Batch fetch PGNs for all sample game_ids:
     ```python
     game_ids = [row.game_id for row in sample_rows]
     pgn_stmt = select(Game.id, Game.pgn).where(Game.id.in_(game_ids))
     pgn_result = await session.execute(pgn_stmt)
     pgn_map = {row.id: row.pgn for row in pgn_result.all()}
     ```
   - For each sample, replay PGN to target ply and extract board_fen:
     ```python
     import io
     result_fens: dict[int, str] = {}
     for row in sample_rows:
         pgn_str = pgn_map.get(row.game_id)
         if not pgn_str:
             continue
         game = chess.pgn.read_game(io.StringIO(pgn_str))
         if not game:
             continue
         board = game.board()
         for i, move in enumerate(game.mainline_moves()):
             board.push(move)
             if i + 1 == row.ply:  # ply is 0-indexed from initial position
                 break
         result_fens[row.full_hash] = board.board_fen()
     return result_fens
     ```

   **IMPORTANT ply semantics note:** ply=0 is the initial position. After the first move, ply=1. So to reach ply N, push N moves. `enumerate(game.mainline_moves())` gives (0, first_move), (1, second_move), etc. To reach position at ply N, push N moves (i.e., break when `i + 1 == row.ply` which means we've pushed `row.ply` moves). Verify this matches the actual data by checking test results.

7. Build NextMoveEntry list:
   ```python
   moves: list[NextMoveEntry] = []
   for row in move_rows:
       gc = row.game_count
       w, d, l = row.wins, row.draws, row.losses
       wp = round(w / gc * 100, 1) if gc > 0 else 0.0
       dp = round(d / gc * 100, 1) if gc > 0 else 0.0
       lp = round(l / gc * 100, 1) if gc > 0 else 0.0
       moves.append(NextMoveEntry(
           move_san=row.move_san,
           game_count=gc, wins=w, draws=d, losses=l,
           win_pct=wp, draw_pct=dp, loss_pct=lp,
           result_hash=str(row.result_hash),
           result_fen=result_fens.get(row.result_hash, ""),
           transposition_count=trans_counts.get(row.result_hash, gc),
       ))
   ```

8. Sort:
   ```python
   if request.sort_by == "win_rate":
       moves.sort(key=lambda m: m.win_pct, reverse=True)
   else:  # frequency (default)
       moves.sort(key=lambda m: m.game_count, reverse=True)
   ```
