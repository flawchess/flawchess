**1. Data isolation fix** (`frontend/src/hooks/useAuth.ts`):
Import `useQueryClient` from `@tanstack/react-query` and get the queryClient instance inside the `AuthProvider` component. Then update the `logout` function to call `queryClient.clear()` before redirecting:

```typescript
import { useQueryClient } from '@tanstack/react-query';

// Inside AuthProvider component body:
const queryClient = useQueryClient();

const logout = (): void => {
    queryClient.clear();
    localStorage.removeItem('auth_token');
    setToken(null);
    setUser(null);
    window.location.href = '/login';
};
```

This ensures that when user A logs out and user B logs in on the same browser, there is no stale cached data from user A's session.
