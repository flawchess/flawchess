    `CREATE DATABASE ... TEMPLATE`; (b) the template is auto-refreshed when the live Alembic head

    safe locally for parallelism (CI stays serial — D-02); the template auto-refreshes on
    Alembic head drift so no manual rebuild is needed after a migration. Keep it terse and
