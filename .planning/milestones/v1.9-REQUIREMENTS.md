# Requirements: FlawChess

**Defined:** 2026-04-09
**Core Value:** Users can determine their success rate for any opening position they specify, filtering by their own pieces only, regardless of how platforms categorize the opening.

## v1.9 Requirements

Requirements for UI/UX Restructuring milestone. Each maps to roadmap phases.

### Openings Desktop Layout

- [x] **DESK-01**: User sees a collapsible sidebar on the left of the Openings page with filter and bookmark icons visible in collapsed state — delivered Phase 49
- [x] **DESK-02**: User can open the sidebar directly to Filters or Bookmarks by clicking the respective icon in the collapsed strip — delivered Phase 49
- [x] **DESK-03**: Only one panel (Filters or Bookmarks) is shown at a time in the sidebar; clicking the other icon switches panels — delivered Phase 49
- [x] **DESK-04**: Filter changes apply live while the sidebar is open (no deferred apply on desktop) — delivered Phase 49
- [x] **DESK-05**: Sidebar overlays the chessboard on smaller desktop screens where a 3-column push layout would be too tight — delivered Phase 49 (overlay/push breakpoint at 1280px)

### Openings Mobile Layout

- [x] **MMOB-01**: Subtab navigation (Moves/Games/Stats) is relocated from its current position — delivered Phase 50-01 (unified control row holding Tabs | Color | Bookmark | Filter inside the sticky wrapper but outside the board collapse region)

### Stats Subtab Layout

- [x] **STAB-01**: "Bookmarked Openings: Results" uses a 2-column layout on desktop (matching "Most Played" style) — delivered Phase 51-02 (lg breakpoint grid with explicit leftRows/rightRows split)
- [x] **STAB-02**: "Most Played Openings as White/Black" uses a stacked layout on mobile (matching bookmarked style) — delivered Phase 51-02 (new MobileMostPlayedRows component)

### Endgames & Games Mobile

- [x] **EGAM-01**: Endgames and Games tab mobile layouts are adjusted consistent with Openings mobile layout changes — delivered Phase 50-02 (44px backdrop-blur sticky row with 44px filter button)

### Homepage

- [x] **HOME-01**: Feature content is visible on desktop without scrolling — delivered Phase 51-03 (static 2-column desktop hero with Opening Explorer preview; pills row removed)

### Global Stats

- [x] **GSTA-01**: Stats page is relabeled to "Global Stats" in navigation and page header — delivered Phase 51-04 (label-to-testid auto-derivation updated desktop nav, mobile bottom bar, More drawer, mobile page header; new page h1)
- [x] **GSTA-02**: More existing filters are enabled on the Global Stats page — delivered Phase 51-01 + 51-04 (opponent_type + opponent_strength wired end-to-end; bot games excluded by default)
