  <action>
    Create `app/services/chesscom_client.py` with:

    ```
    USER_AGENT = "Chessalytics/1.0 (github.com/chessalytics)"
    BASE_URL = "https://api.chess.com/pub/player"

    async def fetch_chesscom_games(
        client: httpx.AsyncClient,
        username: str,
        user_id: int,
        since_timestamp: datetime | None = None,
        on_game_fetched: Callable[[], None] | None = None,
    ) -> AsyncIterator[dict]:
    ```

    Implementation:
    1. Fetch archive list from `{BASE_URL}/{username}/games/archives` with User-Agent header.
    2. On 404 response: raise `ValueError(f"chess.com user '{username}' not found")`.
    3. Iterate through archive URLs. For incremental sync: parse year/month from URL, skip if the month ends before `since_timestamp`.
    4. For each archive: `asyncio.sleep(0.15)` then fetch. On 429: `asyncio.sleep(60)` and retry once.
    5. For each game in the month's response: call `normalize_chesscom_game(game, username, user_id)`. If result is not None, yield it. Call `on_game_fetched()` if provided.

    Helper: `_archive_before_timestamp(archive_url: str, since: datetime) -> bool` — extracts year/month from URL like `.../games/2024/03`, returns True if that month ends before `since`.

    Create `tests/test_chesscom_client.py`:
    - Mock httpx responses using `httpx.MockTransport` or `pytest-httpx` (check if installed, otherwise use `unittest.mock.AsyncMock` to patch `client.get`).
    - Test: valid username returns normalized games from archive.
    - Test: 404 raises ValueError.
    - Test: incremental sync skips old archive months.
    - Test: non-standard variant games are skipped.
    - Test: on_game_fetched callback is called for each game.
    - Mock `asyncio.sleep` to avoid actual delays in tests.
