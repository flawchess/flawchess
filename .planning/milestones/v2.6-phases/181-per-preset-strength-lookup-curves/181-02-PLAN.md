## Locked decisions for this plan

- **D-12 target selection (deterministic, no hand-picked magic numbers):** per preset, pick
  `floor + 100`, the midpoint rounded to the nearest 100, and `ceiling − 100` (from the Plan 01
  lookup `range`). If the preset's range spans fewer than 3 grid steps, emit only the two
  endpoints (2 cells). This yields 2-3 targets per preset, all strictly inside the measured
  curve.
- **D-13 interpolated 95% CI rule (resolves RESEARCH Open Question 1 / Assumption A1):** for a
  predicted off-grid bot_elo that falls INSIDE a merged PAVA block (a plateau), pool the merged
  member cells' `ci_vs_maia` bounds by inverse-variance weighting (the same technique
  `combine_preset_g_preset` uses in calibration_anchor_fit.py). For a bot_elo that falls between
  two DISTINCT blocks, linearly interpolate each bound between the two neighboring blocks'
  bounds. Document this rule verbatim in the generator docstring — D-13 pass/fail is only as
  honest as this rule.
