<objective>
Make bot games invisible to default analytics but visible to the Library games
list, and fix the rating double-conversion landmine that Plan 01's Platform
extension exposes.

insights by default) and STORE-01's backend half (the Library games query can
return flawchess games). Both hinge on the single shared filter `apply_game_filters`
plus one opt-in call site and one rating guard — no per-router scatter (D-02/D-03).

a flawchess opt-in in `get_library_games`; a `platform=='flawchess'` guard in
`_build_card`.
</objective>
