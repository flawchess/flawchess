## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| localStorage â†’ app (read) | Untrusted: the value can be hand-edited in devtools, corrupted by a browser extension, truncated by a storage failure, or written by a DIFFERENT user on a shared browser. Every read must be validated. |
| app â†’ localStorage (write) | Storage can reject a write (QuotaExceededError, Safari private mode). A write failure must degrade, never crash the game. |
