### 2. D-01 — think time IS billed
steps: Start a fresh game, play one move, sit ~30s on your own turn, close the tab, reopen, Resume.
expected: Your clock is ~30s lower than at the start of that turn. (Thinking is billed; being away is not.)
why_human: Real wall-clock delay plus real tab lifecycle events.
result: pass

### 3. D-02 — the bot's interrupted think is REFUNDED
steps: Close the tab while the bot is thinking. Reopen and Resume.
expected: The bot's clock is unchanged from its last committed move. (Its search died with the workers, so we do not bill for thrown-away work.)
why_human: Requires a real in-flight Web Worker search interrupted by a real tab close.
result: pass

### 4. D-05 — Discard confirms first
steps: With a snapshot present, click Discard. Try Cancel, then try Confirm.
expected: A confirmation appears before anything is cleared. Cancel leaves the game intact. Confirm clears the gate and starts a fresh game, with no gate on the next reload.
why_human: Component logic is covered by ResumeGate.test.tsx (passing); this item covers real-browser dialog interaction and reload behavior.
result: pass

### 5. SC2 — an abandoned game leaves no server trace
