  <behavior>
    - applyIncrementMs(remainingMs, incrementMs): returns remainingMs + incrementMs (Fischer). Never negative.
    - computeElapsedMs(anchorMs, nowMs): returns nowMs - anchorMs; recomputed from timestamps, never accumulated tick counts.
    - shiftAnchorForPause(anchorMs, pausedForMs): returns anchorMs + pausedForMs (charges zero time for the hidden interval — PLAY-04).
    - computeSyntheticDebitMs(botRemainingMs, incrementMs): returns Math.round(botRemainingMs / SYNTHETIC_DEBIT_DIVISOR) + Math.round(incrementMs * SYNTHETIC_INCREMENT_SHARE) (168.5 D-02 fraction-of-remaining).
    - reconcileBotDebitMs(realElapsedMs, syntheticMs, botRemainingMs): returns Math.min(Math.max(realElapsedMs, syntheticMs), botRemainingMs - NEVER_FLAG_FLOOR_MS) (D-05 max-then-clamp; result is always < botRemainingMs so the bot never flags).
    - computeRevealDelayMs(rng): returns REVEAL_DELAY_MIN_MS + rng() * (REVEAL_DELAY_MAX_MS - REVEAL_DELAY_MIN_MS) (D-03 floor, ~0.5-1.5s ballpark locked by 168.5).
    - isLowTime(remainingMs): remainingMs < LOW_TIME_THRESHOLD_MS.
    - formatClockLabel(remainingMs): returns "m:ss" normally, "m:ss.d" (tenths) when isLowTime is true (D-07). Clamps negative to "0:00".
  </behavior>
