- **GAP 1 (169-REVIEW.md CR-02, amended ROADMAP SC1, D-15/D-16/D-18):** `commitMove`

  ONLY on the resume edge (`shiftAnchorForPause`). Neither the tick's flag check nor

  Workers keep searching while backgrounded, so the think usually resolves before the
  user returns to the tab).

the tick during a hidden period, which removes the tick as a flag detector and makes the
commit-time clamp the *only* remaining backdoor for that path.
