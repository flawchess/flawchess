## What Was Built

### Task 1: scripts/seed_cohort_cdf.py (NEW)

Near-direct mirror of `scripts/seed_openings.py`. Key properties:

- Module docstring with both local (`uv run python -m scripts.seed_cohort_cdf`) and
  in-container prod usage (`docker compose exec backend /app/.venv/bin/python -m scripts.seed_cohort_cdf`).
- Runbook note: fresh prod DB suppresses percentile chips until this seed runs (graceful degradation,
  mirrors openings pattern).
- `TSV_PATH` points to `app/data/cohort_cdf.tsv` relative to the script.
- `csv.DictReader(f, delimiter="\t")` streaming loop -- memory-efficient for 122k rows.
- `_parse_row()` helper casts TSV strings to Python types: `anchor_elo`/`percentile` -> int,
  `n_users` -> int | None (empty -> None), `value` -> float, `tc`/`metric`/`snapshot_month` -> str.
- `text(_INSERT_SQL)` with bound `:params` -- never f-string interpolation (T-99.1-05 closed).
- `ON CONFLICT (metric, anchor_elo, tc, percentile) DO UPDATE` for idempotent upserts.
- Dev DB seeded: **122,661 rows upserted**, 0 errors. Second run: same count (idempotent).

### Task 2: app/repositories/benchmark_cohort_cdf_repository.py (NEW, D-04 sole async DB access)

```python
async def load_cohort_cells(
    session: AsyncSession,
    anchors: Sequence[int],
    tcs: Sequence[TimeControlBucket],
) -> dict[tuple[CdfMetricId, int, TimeControlBucket], CdfTable]:
```

Design:
- **One SELECT** covering all requested `(metric, anchor_elo, tc)` cells via `anchor_elo.in_(anchors)` and
  `tc.in_(tcs)` (D-03 batched prefetch -- zero extra DB round-trips during the import loop).
- `ORDER BY metric, anchor_elo, tc, percentile` is load-bearing: `CdfTable.breakpoints` is indexed
  parallel to `BREAKPOINT_PERCENTILES` (p1..p99), so ascending percentile order from the DB is required.
- `_group_rows_into_cells()` extracts values into `defaultdict(list)` per `(metric, anchor_elo, tc)` key,
  then builds `CdfTable(breakpoints=tuple(values), n_users=..., snapshot_month=...)`.
- Cells with < 99 rows skipped (defensive guard).
- NULL `n_users` coalesced to 0 (CdfTable.n_users is typed `int`; parity unaffected -- audit column only).
- Empty anchors or tcs: early-return `{}` without querying (T-99.1-07 DoS guard).
- No `asyncio.gather` (single session, CLAUDE.md constraint).

### Task 3: bin/run_local.sh (MODIFIED)

Added block immediately after the openings seed block, before the Stockfish block:

```bash
# Seed cohort CDF if table is empty (first-time setup only — skips instantly otherwise)
COHORT_CDF_COUNT=$(PGPASSWORD=postgres psql -h localhost -p 5432 -U postgres -d flawchess -tAc "SELECT COUNT(*) FROM benchmark_cohort_cdf" 2>/dev/null || echo "0")
if [ "$COHORT_CDF_COUNT" -eq 0 ]; then
  echo "Seeding cohort CDF table..."
  uv run python -m scripts.seed_cohort_cdf
fi
```

Same psql connection params, `|| echo "0"` fallback, and `-eq 0` gate as the openings block.
