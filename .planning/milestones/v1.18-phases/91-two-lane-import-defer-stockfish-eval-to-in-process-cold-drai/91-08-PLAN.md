      1. Refuse if `--api-base` looks prod (without `--allow-prod`).

      3. POST `/imports` twice in parallel: once for chess.com, once for lichess. Capture both job IDs.

         - Peak pg memory ≤ `POSTGRES_MEMORY_MAX_MB`? Else FAIL.
         - Peak swap_pct ≤ `SWAP_USAGE_MAX_PCT`? Else FAIL.
