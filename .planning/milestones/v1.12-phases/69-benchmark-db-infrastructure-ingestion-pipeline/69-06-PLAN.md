WITH side_games AS (
  SELECT
    CASE WHEN g.user_color = 'white' THEN g.white_rating ELSE g.black_rating END AS player_rating,
    g.time_control_bucket
  FROM games g
)
SELECT
  CASE
    WHEN player_rating < 800 THEN NULL
    WHEN player_rating < 1200 THEN 800
    WHEN player_rating < 1600 THEN 1200
    WHEN player_rating < 2000 THEN 1600
    WHEN player_rating < 2400 THEN 2000
    ELSE 2400
  END AS rating_bucket,
  time_control_bucket AS tc_bucket,
  count(*) AS games
FROM side_games
WHERE player_rating IS NOT NULL
GROUP BY 1, 2
ORDER BY 1, 2;

-- Eval coverage per cell (Phase 70 dependency)
SELECT
  CASE
    WHEN coalesce(g.white_rating, g.black_rating) < 800 THEN NULL
    WHEN coalesce(g.white_rating, g.black_rating) < 1200 THEN 800
    WHEN coalesce(g.white_rating, g.black_rating) < 1600 THEN 1200
    WHEN coalesce(g.white_rating, g.black_rating) < 2000 THEN 1600
    WHEN coalesce(g.white_rating, g.black_rating) < 2400 THEN 2000

  count(gp.*) FILTER (WHERE gp.eval_cp IS NOT NULL OR gp.eval_mate IS NOT NULL) AS positions_with_eval
FROM games g JOIN game_positions gp ON gp.game_id = g.id
GROUP BY 1, 2

The FlawChess import pipeline stores this in `game_positions.eval_cp` as `SmallInteger`.

- Game: `https://lichess.org/<platform_game_id>` (Lichess game id `<platform_game_id>`)

### Mate annotations

`[%eval #4]` indicates White mates in 4. python-chess parses this so that `pov.white().score()` returns `None` and `pov.white().mate()` returns `4`. FlawChess stores the mate value in `game_positions.eval_mate` as `SmallInteger` (signed: positive = White mates, negative = Black mates).
