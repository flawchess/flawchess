<objective>
Lay the schema + engine-API foundation for the all-ply eval pipeline: the node-budget
Stockfish call (EVAL-02), the full-analysis completion marker column + its pending-pick
index (EVAL-05), the cross-user opening dedup index (EVAL-03), and the verified backfill
that seeds the dedup source set from already-fully-evaluated games (D-116-06).

Purpose: These are the contracts the drain coroutine (Plan 02) and the memory accounting
(Plan 03) build on — the engine call signature, the column/indexes, and the migration must
exist before the drain can pick games, dedup, evaluate, and mark them.

Output: `evaluate_nodes()` in engine.py, `full_evals_completed_at` on games, two new
indexes, one Alembic migration with a verified backfill, plus Wave-0 test scaffolds for the
engine call (EVAL-02) and the migration (EVAL-05).
</objective>
