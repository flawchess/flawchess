## What Was Built

### Categorical Color Scheme (arrowColor.ts)

Replaced the oklch gradient interpolation with 10 named hex color constants and a simple threshold-based function:

- `DARK_GREEN` / `DARK_GREEN_HOVER` — win rate 60%+
- `LIGHT_GREEN` / `LIGHT_GREEN_HOVER` — win rate 55-60%
- `GREY` / `GREY_HOVER` — neutral zone 45-55%
- `LIGHT_RED` / `LIGHT_RED_HOVER` — loss rate 55-60%
- `DARK_RED` / `DARK_RED_HOVER` — loss rate 60%+

Moves with fewer than 10 games always show grey. When both win and loss rates exceed 55%, the higher rate wins (ties go to green/win).

### Fixed Arrow Render Order (ChessBoard.tsx)

Sort was previously ascending by arrowSortKey (green=0 drawn first = behind grey). Fixed to descending sort so grey (key=2) renders first (bottom of SVG) and colored arrows (key=0,1) render last (on top).

### Updated Popover Text (Openings.tsx)

Both desktop and mobile info popovers now describe the categorical scheme instead of the gradient description.
