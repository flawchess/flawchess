# Quick Task 260710-x3d

## Goal

Replace the analysis board's `?fen=` free-play entry point with an opening-moves
main line carried as a `?line=` URL param (comma-separated UCI moves from the
standard start position). This lets the user walk all the way back to move 1 in
the analysis board's variation tree instead of landing on a bare snapshot FEN.

- Cursor opens at the **end** of the line (already `loadMainLine`'s default).
- **Remove** the `fen` param entirely (param, guard, builder, remount key, tests).
- **Keep** game mode (`?game_id=&ply=`) — it loads imported games with
  flaw/tactic sidelines, eval chart and auto-flip; not replaceable by a URL move list.

## Design decisions

- **UCI, not SAN, in the URL.** UCI tokens (`e2e4`, `e1g1`, `e7e8q`) are URL-safe
  with no encoding; SAN drags in `+`, `#`, `=`, `O-O`. Openings holds SAN
  (`chess.moveHistory`), so the URL builder converts SAN→UCI and the reader
  converts UCI→SAN (feeding the existing `loadMainLine(sans, STARTING_FEN)`).
- **Root is always `STARTING_FEN`.** Opening lines start from the standard
  position; `loadMainLine` sets `rootFen` when it seeds.
- **Malformed `line` degrades gracefully** — the reader stops at the first
  illegal UCI move (mirrors `loadMainLine`'s existing "stop on illegal SAN"
  tolerance), same defensive posture the old FEN guard had.

### Task 1 — `lib/analysisUrl.ts`: swap FEN helpers for line helpers
- **files:** `frontend/src/lib/analysisUrl.ts`, `frontend/src/lib/analysisUrl.test.ts`
- **action:**
  - Remove `FEN_PARAM` and `buildAnalysisUrl`.
  - Add `const LINE_PARAM = 'line'`.
  - Add `buildAnalysisLineUrl(sans: string[]): string` — replays SAN on a fresh
    `new Chess()`, emits `from+to+(promotion ?? '')` per move, joins with `,`.
    Empty input → `/analysis` (bare board). Stops on illegal SAN.
  - Add `parseAnalysisLineParam(lineParam: string | null): string[]` — splits on
    `,`, replays each UCI token (`from=slice(0,2)`, `to=slice(2,4)`,
    `promotion=slice(4,5) || undefined`) via `chess.move(...)`, returns the SAN
    array; breaks (stops) on the first illegal/unparseable token. `null`/empty → `[]`.
  - Import `{ Chess } from 'chess.js'`; update the module docstring to cover
    move-list↔URL encoding.
  - Keep `buildGameAnalysisUrl` unchanged.
  - Rewrite `analysisUrl.test.ts`: drop `buildAnalysisUrl` cases; add
    `buildAnalysisLineUrl` (start→`/analysis`, a line → correct UCI join, chars
    URL-safe) and `parseAnalysisLineParam` (round-trip via buildAnalysisLineUrl,
    empty/null → `[]`, illegal token stops, promotion `e7e8q`).
