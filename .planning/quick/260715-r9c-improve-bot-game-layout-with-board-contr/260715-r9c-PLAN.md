# Quick Task 260715-r9c: Improve bot game layout

Improve the `/bots` play-vs-bot layout (`frontend/src/pages/Bots.tsx`).

## Decisions (from user + clarifying question)

- **Desktop 2-column layout**: clocks STAY in the right column (with move list +
  Resign/Draw). Board-width clock strips apply to the single-column layout only.
- **Board controls** (reset / back / forward / flip): reuse the existing
  `components/board/BoardControls.tsx`, placed directly below the board in both
  layouts.
  - back/forward/reset drive the hook's view-only `viewPly` cursor
    (`viewedPly`/`liveGamePly`/`viewPly` already exposed by `useBotGame`).
  - reset → `viewPly(0)`; back → `viewPly(viewedPly-1)`; forward →
    `viewPly(viewedPly+1)`.
  - flip → a new local `flipped` toggle (initialized from
    `settings.userColor === 'black'`), replacing the derived-only orientation.
