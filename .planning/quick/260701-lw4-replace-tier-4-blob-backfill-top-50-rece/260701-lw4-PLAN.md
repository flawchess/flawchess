<objective>
Replace the tier-4 blob-backfill top-50 recency window in `_claim_tier4_blob` with a
two-stage Efraimidis–Spirakis (ES) weighted lottery mirroring tier-3's
`_claim_tier3_derived`. This is a fairness/ordering fix, NOT a throughput fix: today the
window is a hard cutoff (game #51 by `full_evals_completed_at DESC` has probability zero),
so an old analyzed corpus (e.g. user 28's ~5k games) never drains while a trickle of
freshly-analyzed games keeps refilling the window. The ES floor term gives every
pending-blob game nonzero mass so the whole corpus drains; recency weighting keeps fresh
games dominant.

Purpose: unblock draining of the full flaw-blob backfill backlog without starving old games.
Output: rewritten `_claim_tier4_blob`, 4 new tunable tier-4 constants, updated stale Phase
146 D-01 comments, and a replaced test asserting both anti-starvation and recency preference.
</objective>

Stage 1 — pick a non-guest user weighted by last_activity recency + floor:
SELECT u.id FROM users u WHERE u.is_guest = false (KEEP the existing
"guests excluded (QUEUE-08)" intent comment on that line) AND EXISTS a game g for that user
with g.full_evals_completed_at IS NOT NULL joined to a game_flaws row gf where
gf.game_id = g.id AND gf.allowed_pv_lines IS NULL. ORDER BY
-ln(random()) / (exp(-EXTRACT(EPOCH FROM (now() - COALESCE(u.last_activity, '1970-01-01'::timestamptz))) / :tau_u) + :floor_u)
LIMIT 1. Bind :tau_u and :floor_u via the params dict. `.one_or_none()`; if None, return None.

Stage 2 — pick that user's game weighted by full_evals_completed_at recency + floor
(NO tc_multiplier CASE — blob enrichment is not time-control sensitive):
SELECT g.id FROM games g WHERE g.user_id = :picked_user AND g.full_evals_completed_at IS NOT NULL
AND EXISTS a game_flaws gf where gf.game_id = g.id AND gf.allowed_pv_lines IS NULL. ORDER BY
-ln(random()) / (exp(-EXTRACT(EPOCH FROM (now() - COALESCE(g.full_evals_completed_at, '1970-01-01'::timestamptz))) / :tau_g) + :floor_g)
LIMIT 1. Bind :picked_user, :tau_g, :floor_g via the params dict. If None, return None
(fall through), mirroring tier-3.
