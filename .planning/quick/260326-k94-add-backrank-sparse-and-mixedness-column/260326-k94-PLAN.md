    1. Add `_compute_backrank_sparse(board: chess.Board) -> bool`:

       - `BACKRANK_SPARSE_THRESHOLD = 4` — named constant.
       - `white_backrank_count = bin(board.occupied_co[chess.WHITE] & chess.BB_RANK_1).count('1')`

    2. Add `_compute_mixedness(board: chess.Board) -> int`:
       - Precompute `_MIXEDNESS_REGIONS` at module level: list of (mask, y) tuples.
         `_SMALL_SQUARE = 0x0303` (2x2 region at a1-b1-a2-b2).
         For y_idx in range(7), for x_idx in range(7): mask = _SMALL_SQUARE << (x_idx + 8 * y_idx), y = y_idx + 1.
         Total: 49 regions.
       - Precompute `_MIXEDNESS_SCORE_TABLE`: a dict keyed by (y, white_count, black_count) -> int score.
         Transcribe the score function from Divider.scala exactly:
         ```
         def _mixedness_score(y: int, white: int, black: int) -> int:
             match (white, black):
                 case (0, 0): return 0
                 case (1, 0): return 1 + (8 - y)
                 case (2, 0): return (2 + (y - 2)) if y > 2 else 0
                 case (3, 0): return (3 + (y - 1)) if y > 1 else 0
                 case (4, 0): return (3 + (y - 1)) if y > 1 else 0
                 case (0, 1): return 1 + y
                 case (1, 1): return 5 + abs(4 - y)
                 case (2, 1): return 4 + (y - 1)
                 case (3, 1): return 5 + (y - 1)
                 case (0, 2): return (2 + (6 - y)) if y < 6 else 0
                 case (1, 2): return 4 + (7 - y)
                 case (2, 2): return 7
                 case (0, 3): return (3 + (7 - y)) if y < 7 else 0
                 case (1, 3): return 5 + (7 - y)
                 case (0, 4): return (3 + (7 - y)) if y < 7 else 0
                 case _: return 0
         ```
         Use `_mixedness_score` as a helper function (not a table lookup — the match statement is clear enough).
       - `_compute_mixedness`: iterate `_MIXEDNESS_REGIONS`, for each (mask, y): white_count = popcount of (board.occupied_co[chess.WHITE] & mask), black_count = popcount of (board.occupied_co[chess.BLACK] & mask), accumulate `_mixedness_score(y, white_count, black_count)`.
       - Use `bin(bb).count('1')` for popcount (python-chess `chess.popcount` may not exist in all versions).
