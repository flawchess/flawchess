    MetricStatTooltip (pure markup, no popover plumbing):
    - Props (exact shape):
      `{ name: string; explanation: ReactNode; value: number; baseline: 0 | 0.5 | number;
         unit: 'percent' | 'pawns'; gameCount: number; level: ConfidenceLevel; pValue: number | null;
         vocabulary: 'score' | 'eval'; neutralLower: number; neutralUpper: number;
         baselineLabel: string; methodology: ReactNode; lastPlayedAt?: string | null; }`
    - Imports `ConfidenceLevel` from `@/lib/scoreConfidence`.
    - Paragraph 1 (bold name + explanation): `<p><strong>{name}.</strong> {explanation}</p>`. Single paragraph,
      no link, no "see ..." cross-reference.
    - Paragraph 2 (value line):
        unit === 'percent':
          - valuePct = (value * 100).toFixed(1) + '%'
          - diffPct  = Math.abs(value * 100 - baseline * 100).toFixed(1) + '%'
          - direction = value >= baseline ? 'above' : 'below'
          - signed prefix only when baseline === 0 (gap metrics): sign = value >= 0 ? '+' : ''
            For score-vs-50% (baseline === 0.5) the value is unsigned (e.g. "52.3%").
          - If diffPct === '0.0%': `"{sign}{valuePct} {name} over {gameCount} games, at the {baselineLabel} baseline."`
          - Else: `"{sign}{valuePct} {name} over {gameCount} games, {diffPct} {direction} the {baselineLabel} baseline."`
        unit === 'pawns':
          - signed = (value >= 0 ? '+' : '') + value.toFixed(2)
          - `"{signed} pawns over {gameCount} games."` (no baseline-distance text — matches existing EvalConfidenceTooltip)
    - Paragraph 3 (verdict + confidence headline + p-value):
        pickVerdict (switch on `vocabulary`):
          - 'score': value >= neutralUpper -> 'strength'; value <= neutralLower -> 'weakness'; else 'difference'
          - 'eval':  value >= neutralUpper -> 'advantage'; value <= neutralLower -> 'disadvantage'; else 'deviation'
        headline:
          - level === 'low'                                    -> 'Inconclusive.'
          - verdict in {'strength','weakness','advantage','disadvantage'} -> `${lead} a real ${verdict}.`
          - verdict === 'difference'                           -> `${lead} a real difference from the {baselineLabel} baseline.`
          - verdict === 'deviation'                            -> `${lead} a real deviation from {baselineLabel}.`
          where lead = level === 'high' ? 'Likely' : 'Possibly'.
        confidence tail: `"{Low|Medium|High} confidence{ (p = X.XXX)}.".` Omit ` (p = X.XXX)` when pValue === null.
    - Optional lastPlayedAt paragraph (parity with WdlConfidenceTooltip): inserted between paragraph 3 and methodology
      when `lastPlayedAt` is a non-empty string. Use `formatRelativeDate` + `formatAbsoluteDate` from `@/lib/relativeDate`
      following the exact pattern in WdlConfidenceTooltip.
    - Paragraph 4 (methodology): `<p className="opacity-70 italic">{methodology}</p>`.
    - Container: `<div className="text-left space-y-1">` (matches existing popovers).
