# Quick Task 260705-bm3 — Phase 151 Maia UAT feedback

## Goal

page + homepage acknowledgements.

## Decisions (locked)

- **ELO range → compute the full 600–2600** (user pick). maiachess.com itself
  displays 600–2600, so we expand the actual Maia ladder (step 100 → 21 rungs), not
  just the chart axis. 600–1000 / 2100–2600 are extrapolation beyond the phase-151
  contract's behaviorally-validated 1100–2000 band, but they match maiachess.com's
  own presented range. Slider + eval bar widen to match (inherent, both derive from
  the ladder).
- **Legal box removed, attribution compacted into the header tooltip.** The tooltip
  keeps the maia3 source link (github.com/CSSLab/maia3), i.e. the AGPL offer-source
  in shortened form; the full AGPL license text moves out of the UI (README precedent
  retains the formal notice). Flagged as a compliance note, applied per explicit UAT.
