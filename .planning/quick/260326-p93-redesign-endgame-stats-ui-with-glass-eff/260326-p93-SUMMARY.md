## Changes Made

### EndgameWDLChart.tsx

- Replaced hardcoded oklch color strings with imported `WDL_WIN`, `WDL_DRAW`, `WDL_LOSS` constants from `WDLBar.tsx`
- Added `GLASS_OVERLAY` constant and applied `backgroundImage: GLASS_OVERLAY` to each WDL bar segment
- Added grey-outlined game count bar below WDL bar (proportional to `maxTotal`)
- Added `Conv` and `Recv` mini-bars (h-3) with label + bar + percentage text, rendered only when games > 0
- Added `ENDGAME_TYPE_DESCRIPTIONS` map and per-row `InfoPopover` on each endgame type label
- Reorganized header tooltip into 4 `<p>` paragraphs with spacing
- Added `onSelectedCategoryClick` prop; onClick dispatches to the right handler based on `isSelected`
- Removed unused Recharts imports (`ChartContainer`, `ChartTooltip`, `ChartLegend`, `ChartLegendContent`, `BarChart`, `Bar`, `XAxis`, `YAxis`, `CartesianGrid`) and the hidden `sr-only` chart block
- All interactive elements have `data-testid` attributes

### Endgames.tsx

- `handleCategoryClick` simplified to always select (removed deselect-on-same-row logic)
- Added `handleSelectedCategoryClick` that calls `navigate('/endgames/games')`
- Both `EndgameWDLChart` instances (desktop and mobile) updated to pass `onSelectedCategoryClick`
