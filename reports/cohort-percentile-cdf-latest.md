If coverage in the table above is too sparse for the shipping goal (per CONTEXT D-11 strict-default policy), the operator can choose:

- **Option A — ship strict defaults**: chip suppresses for users in the affected (metric, anchor, TC) cells; ``interpolate_cohort_percentile`` returns ``None`` → frontend renders nothing.
